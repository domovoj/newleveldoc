Главный шаблон

Место нахождения

main.tpl - главный шаблон сайта находиться в корне папки шаблона.

Использование

Используется как обертка для других шаблонов: (Шаблоны Shop и Шаблоны сайта) и Компоненты шаблонов

Компоненты

  • Хедер {include_tpl('header')}

  • Главное меню категорий {\Category\RenderMenu::create()->setConfig(array('cache'=>TRUE))->load('category_menu')}

  • {$content}

    В ету переменную вставляються (Шаблоны Shop и Шаблоны сайта)

  • Футер {include_tpl('footer')}

  • Юзерский тулбар {include_tpl('user_toolbar')}

  • Конфигурационный файл {include_tpl('config.js')}

    Этот шаблон содержыт JS - переменные, которые связанны с:

    • PHP переменными
    • Найстроками БД
    • Лангами
    • Настройками шаблона

    Эти и другие переменные позицинируются в шаблоне как "Глобальные переменые"

  • Файл JS - шаблонов {include_shop_tpl('js_templates')}

    Этот шаблон содержыт JS - шаблоны библиотекы underscore.js и розметки, такие как:

    • Шаблон корзины
    • Шаблон корзины оформления заказа
    • Шаблон автозаполнений поиска
    • Шаблон собщения о появлении
    • Розметку Tooltip
    • Розметку попапу - количества нейденых профильтрированных свойств в фильтре
    • Розметку попапу модальнных уведомлений
    • Розметку попапу моального подтвержения действия

  • подробно о функции downloadJSAtOnload()

    Это JS - функция которая загружает JS - файлы на событие onload документа
