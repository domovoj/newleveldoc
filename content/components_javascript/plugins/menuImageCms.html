Плагин menuImageCms

Место нахождения

Файл _jquery.imagecms.shop.js

Использование

Используэться для Главного меню категорий

Описание

Плагин поддержывает touch устройства, параметры evLF и evLS (которые отвечают за событие открытия фреймов вторго и третьего уровней) автоматически принимают значение toggle

Рекомендуема структура для основных блоков меню:

<li> //по єтому єлементу будет обрабатываться событие (toggle или hover) которое покажет выпадающий блок или же перейдет по силке если вложенных категорий нет

  <a>…</a> //этот элемент это ссылка на определенную категорию, он будет участвовать в выборе активного пункта по всей иерархии

  <div> //выпадающий блок
   …
  </div>
</li>

Плагин определяет по какой стороне будет позиционироватся блок, он дает класы left-drop и right-drop для выпадающего фрейма 2-ых уровней категорий и задает значение свойств left и right соотвественно,

Плагин добавляет клас is-side для фрейма 3-го уровня категорий, посмотреть пример

Свойства

Название параметра значение по умолчанию Описание
item $(this).find('li:first') Jquery обьэкт посредством которого будут открыватся вложенные подкатегории, а также позиционироваться
direction null Может быть двох значений left или right. Определяет по кокой стороне выравнивать открытые вложенные подкатегории
effectOn 'fadeIn' Задает строковое название ефекта открытия второго уровня категорий
effectOff 'fadeOut' Задает строковое название ефекта закрытия второго уровня категорий
effectOnS 'fadeIn' Задает строковое название ефекта открытия третего уровня категорий, будет задействован если задан параметр sub2Frame
effectOffS 'fadeOut' Задает строковое название ефекта закрытия третего уровня категорий, будет задействован если задан параметр sub2Frame
duration 0 Задержка перед открытием вторго уровня категорий, она принимает значение "0" если идет перемещение между соседними пунктами категорий
drop 'li > ul' Селектор фрейма второго уровня категорий, он должен быть вложен в элемент заданый в параметре item
countColumn 'none' Количество колонок второго уровня категорий (нужно указывать максимальное количество), если параметр sub2Frame со значением null, то меню будет иметь вид
Сортировка категорий, номера колонок которых выбранные в админ
columnPart false Опердиляет нужно ли сортировать категории 2-го уровня
columnPart2 false Опердиляет нужно ли сортировать категории 3-го уровня
columnClassPref 'column_' Префикс названия класа для пункта второго уровня категорий
columnClassPref2 'column2_' Префикс названия класа для пункта третего уровня категорий
sub3Frame 'ul ul' Селектор фрейма 3-го уровня категорий, селектор должен быть вложен в элемент заданый в параметре drop
sumx 10 Максимальное количество колонок (размеры колонок заданы в стилях подробно)
durationOn 0 Продолжытельнось открытия фрейма 2-го уровня категорий
durationOff 0 Продолжытельнось закрытие фрейма 2-го уровня категорий
durationOnS 0 Продолжытельнось открытия фрейма 3-го уровня категорий, будет задействован если задан параметр sub2Frame
durationOffS 0 Продолжытельнось закрытие фрейма 3-го уровня категорий, будет задействован если задан параметр sub2Frame
animatesub3 false Опредиляет нужно ли при открытии 3-го уровня категорий использовать анимацию для задания размеров общиг враперов 2-го и 3-го уровней
dropWidth null Определяет шырину выпадающей менюшки 2-го уровня категорий
sub2Frame null Если указано то 3-й уровень будет выпадать слева, справа, или даже вниз если параметр sub2Frame со значением null, то меню будет иметь вид
evLF 'hover' Событие по которму будет открыватся и закрыватся фрейм 2-го уровня категорий
evLS 'hover' Событие по которму будет открыватся и закрыватся фрейм 3-го уровня категорий
hM 'hoverM' Этот клас будет присваивается по наведению или клику (параметры evLF и evLS) на елемет для которого вызывается плагин, на пункт меню первого уровня и второго уровня
vertical false Опредиляет вертикальное меню пример вертикального меню
Кэш
menuCache false Если этот параметр true то активный пункт определяется посредством плагина
activeFl activeClass селектор сылки первого уровня категорий
parentTl 'li' Слейдущий соседний селектор сылки второго уровня категорий
otherPage undefined Используется для выбора активного пункта если пользователь находится на странице товара должен содержать полный адрес к товару через все категории
