optionsMenu

Место нахождения

Файл _scripts.js

Использование

Обект optionsMenu используется для задания параметров плагина,

а также при инициализации плагина menuImageCms

$('.menu-main').menuImageCms(optionsMenu);

При ресайзе окна браузера используется метод resize для переинициализации

$('.menu-main').menuImageCms('refresh');

Свойства

Все свойства этого обекта, ето параметры плагина menuImageCms
