Главное меню категорий 2-й уровень

Главное меню категоий может быть двох видов:

Пример меню с по строчным выводом категорий 2-го уровня
HTML:
<div class="menu-main not-js menu-row-category"></div>
JS:
var optionsMenu = {
    //countColumn: 5
    sub2Frame: '.frame-l2', //селектор фрейма 3-го уровня меню категорий
    dropWidth: 475 //не обезательнный параметр который указывает на шырину фрейма 2-го уровня категорий
}
Пример меню с по столбиковым выводом категорий 2-го уровня
HTML:
<div class="menu-main not-js menu-col-category"></div>
JS:
var optionsMenu = {
    countColumn: 5 //необезательный параметр который указывает на количество категорий 2-го уровня
    //sub2Frame: '.frame-l2',
    //dropWidth: 475
}
