Компоненты

Компонент HTML:

Визуально область этого блока, выглядит так:

  1. Верхнее меню {load_menu('top_menu')}
  2. Сылки профиля {include_shop_tpl('auth_data')}
  3. Заказ дзвонка
    <ul class="tabs">
    <li class="btn-order-call">
        <a href="#ordercall" data-drop="#ordercall" data-source="{site_url('shop/callback')}">
            <span class="icon_order_call"></span>
            <span class="text-el d_l">{lang('Заказать звонок','newLevel')}</span>
        </a>
    </li>
</ul>
    • <ul class="tabs">
    <li class="btn-order-call">
        <a href="#ordercall" data-drop="#ordercall" data-source="{site_url('shop/callback')}">

      Такая структура нужна для приминения плагина tabs 

    • data-drop="#ordercall" data-source="{site_url('shop/callback')}"

      Такие атрибуты нужны для применения плагина drop

    Открываеться всплывающое окно колбека при возникновении события "click"

  4. Форма поиска

    <form name="search" method="get" action="{shop_url('search')}">
    <span class="btn-search"> 
        <button type="submit">
            <span class="icon_search"></span> 
            <span class="text-el">{lang('Найти','newLevel')}</span>
        </button>
    </span>
    <div class="frame-search-input">
        <input 
            type="text"
            name="text"
            autocomplete="off"
            value="{if strpos($CI->uri->uri_string, 'search') !== false}{htmlspecialchars($_GET['text'])}{/if}"
            placeholder="{lang('Поиск по сайту', 'newLevel')}"
            class="input-search"
            id="inputString"
            />
        <div id="suggestions" class="drop drop-search"></div> 
    </div>
</form>
    • Атрибут id="inputString" используется для приминение плагина autocomplete подробно о плагине autocomplete
    • Атрибут id="suggestions" используется для вставки содержимого результата исопользование плагина autocomplete подробно о плагине autocomplete
    • В качестве структуры результатов автодополнения выступает часть файла js_templates (скрипт с атрибутом id="searchResultsTemplate")
