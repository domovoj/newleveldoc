Главное меню категорий

Место нахождения

Папка category_menu находиться в корне папки шаблона.

Структура

NewLevel/
├── level_0
│   ├── container.tpl
│   ├── item_default.tpl
│   │
│   ├── item_first.tpl         │
│   ├── item_last.tpl          └── не используються
│   │
│   ├── item_default_active.tpl│
│   ├── item_first_active.tpl  │
│   ├── item_last_active.tpl   └── не используються поскольку активный пункт опредиляется посредством JS, в строке вызова опредиляется как ->setConfig(array('cache'=>TRUE))
├── level_1
├── …
└── level_N

Использование

Используется для отображений категорий
Рендер меню: {\Category\RenderMenu::create()->setConfig(array('cache'=>TRUE))->load('category_menu')}
Находится в главном шаблоне main.tpl

Компоненты

Файл main.tplтам находятся строка:

  • <div class="frame-menu-main [type menu level0]">, где:

    • frame-menu-main - клас стилизации
    • [type menu level0] - может иметь клас (horizontal-menu пример) или (vertical-menu пример)
Первый уровень (level_0)

Файл container.tpl выступает контейнером первого уровня меню, там находятся одна главная строка:

  • <div class="menu-main not-js [type menu level1]"></div>, где клас:

    • menu-main - клас стилизации
    • not-js - клас который удаляется если включен JS
    • [type menu level1] - может иметь клас (menu-row-category пример) или (menu-col-category пример)
Второй уровень (level_1)

Файл item_default.tpl выступает шаблоном для пунктов 2-го уровня категорий меню, там находится одна главная строка:

<li class="column_{echo $CI->load->module('new_level')->getCategoryColumns($id)}"></li>

Выводить номер колонки выбраной в админке:

{echo $CI->load->module('new_level')->getCategoryColumns($id)}

Префикс column_ нужен для сортировки категори по колонках, опредиляется в админке в настройках модуля NewLevel подробнее, используется в JS обекте optionsMenu

Третий уровень (level_2)

Файл item_default.tpl выступает шаблоном для пунктов 3-го уровня категорий меню, там находится одна главная строка:

<li class="column2_{echo $CI->load->module('new_level')->getCategoryColumns($id)}"></li>

Выводить номер колонки выбраной в админке:

{echo $CI->load->module('new_level')->getCategoryColumns($id)}

Префикс column2_ нужен для сортировки категорий по колонках, опредиляется в админке в настройках модуля NewLevel подробнее, используется в JS обекте optionsMenu
